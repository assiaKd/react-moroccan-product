# 🇲🇦 React Moroccan Product

A modern React + TypeScript product listing application inspired by Moroccan-style UI and e-commerce patterns.  
The project allows users to browse products, view details, and manage a wishlist.

---

## ✨ Features

- 🛍️ Product listing grid
- ❤️ Wishlist toggle system
- 🔍 Product filtering and selection
- 📱 Responsive design
- ⚡ Fast React rendering with reusable components
- 🎯 TypeScript type-safe architecture

---

## 🧱 Tech Stack

- React (Vite)
- TypeScript
- CSS (custom styling)
- React Hooks (useState, useEffect)
- Component-based architecture

---

## 📁 Project Structure


src/
